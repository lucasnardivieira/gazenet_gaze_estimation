# GazeNet - Estimativa de Atencao Visual

Projeto de pesquisa para estimativa de direcao do olhar (gaze estimation) em imagens estaticas, com deteccao de pessoas/rostos, fusao com objetos da cena e apoio a analises de Theory of Mind (ToM).

## O que este projeto faz
- Detecta pessoas e rostos na imagem
- Estima para onde cada pessoa esta olhando
- Relaciona o olhar com objetos e outras pessoas na cena
- Gera base estruturada para analise de estados mentais em cenas sociais

## Ambiente recomendado
Este projeto foi feito para rodar melhor no laboratorio Jupyter do LAPIX, onde o ambiente de GPU, dependencias e caminhos de dados ja estao alinhados com o notebook principal.

Arquivo principal de execucao:
- gaze_object_pipeline.ipynb

## Observacao
Fora do ambiente do LAPIX, pode ser necessario ajustar caminhos, credenciais e versoes de pacotes para executar o pipeline completo.
